关于这个中文翻译版

这个站点是 AI Engineering from Scratch 的非官方中文翻译和本地化版本，用 20 个阶段、503 门课程带你亲手构建核心 AI 算法，从 linear algebra 到 autonomous agents，覆盖 Python、TypeScript、Rust 和 Julia。

为什么存在

很多 AI 学习材料是碎片化的：一篇论文、一篇 fine-tuning 文章、一个框架 demo。你可能能上线 chatbot，却解释不清 loss curve；能把工具接进 agent，却不知道调用它的模型里 attention 在做什么。

这套课程提供一条主线。每个算法先从原始数学和代码手写，再对照生产库运行，让你看清框架到底做了什么。每课最后都会留下一个可复用 artifact：prompt、skill、agent 或 MCP server。

中文化范围

中文仓库保留原课程结构、原代码、原提交历史和 MIT License attribution，并补充中文课程正文、中文 quiz、中文目录、中文路线图、中文术语表和 GitHub Pages 站点配置。翻译目标是忠实、可读、便于中文读者跟课，不把原课程改写成另一个原创课程。

站点本身刻意保持简单：手写 HTML、CSS 和 vanilla JavaScript，不依赖前端框架。site/build.js 从仓库中的课程 Markdown 生成目录、搜索索引、sitemap 和 llms.txt，避免发布数字与源码漂移。
