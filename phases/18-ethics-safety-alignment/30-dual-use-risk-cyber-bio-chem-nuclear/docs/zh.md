# Dual-Use Risk — Cyber, Bio, Chem, Nuclear Uplift / 双用途风险：网络、生物、化学与核能力提升

> 2026 年 dual-use 图景，按 domain 看。Bio/chem：Lesson 17 覆盖 WMDP；Anthropic bioweapon-acquisition trial（2.53x uplift）和 OpenAI 2025 年 4 月 Preparedness Framework v2 warning（“on the cusp of meaningfully helping novices create known biological threats”）标志了拐点。Cyber（Anthropic 2025 年 11 月报告）：Chinese-linked state actors 使用 Claude 的 agentic coding tool 自动化最多 90% 的 cyberattack campaign，人类只在 4-6 steps 中介入；OpenAI “trusted access” pilot 为经过审核的 security organisations 提供 capability access，用于 defensive dual-use work。Chem/bio execution gap erosion：经典防御曾是 “information access alone is insufficient”。Vision-enabled frontier models（GPT-5.2、Gemini 3 Pro、Claude Opus 4.5、Grok 4.1）可以观察 wet-lab video 并实时 correction。2025 年 12 月：OpenAI 展示 GPT-5 迭代 wet-lab experiments，通过 AI-driven protocol optimization 达到 79x efficiency improvement。Novice-vs-expert pattern：AI 给 novices 更大的相对 uplift，但给 experts 更大的绝对 capability。

## The Concept / 概念

### Bio/chem uplift narrative / Bio/chem uplift 叙事

三个阶段（为保持连贯，复述 Lesson 17）：

1. **2024 “mild uplift”。** 早期 Preparedness/RSP evaluations 报告 novices 相对 internet search 有小优势。
2. **2025 年 4 月 “on the cusp”。** OpenAI PF v2 警告模型 “on the cusp of meaningfully helping novices create known biological threats.”
3. **2025 Anthropic bioweapon-acquisition trial。** Controlled novice study；acquisition-phase tasks 上 2.53x uplift；不足以 rule out ASL-3。

变化是质性的：“mild” 在 18 个月内演变为 “plausibly enabling”，即使没有单点 capability breakthrough。

### Chem/bio execution-gap erosion / 化学/生物执行鸿沟侵蚀

历史防御：information 必要但不充分；执行 protocol 的技能会阻挡 novices。2025 frontier models with vision 部分击穿这个防御：

- **Real-time protocol correction。** GPT-5.2、Gemini 3 Pro、Claude Opus 4.5、Grok 4.1 能观察 wet-lab video，并在 procedure 中途 flag errors。
- **2025 年 12 月 OpenAI demonstration。** GPT-5 迭代 wet-lab experiments，通过 protocol optimization 实现 79x efficiency improvement。

含义：execution-skill-as-defense 正在被侵蚀。Procurement 和 equipment gaps 仍存在，但 tacit-knowledge gap 在缩小。

### Cyber uplift (November 2025) / Cyber uplift（2025 年 11 月）

Anthropic 2025 年 11 月报告：Chinese-linked state actors 使用 Claude 的 agentic coding tool 自动化 80-90% 的 cyberattack campaign。Human intervention 只需要 4-6 steps。

含义：
- Agentic coding 是 attack-automation primitive。此前 AI cyber assistance 被限制在 code-snippet level；agentic workflows 集成 reconnaissance、exploitation、post-exploitation 和 exfiltration。
- 这 4-6 个 human steps 是 bottleneck；未来 capability gains 会减少这个数字。
- Defensive dual-use：OpenAI 的 “trusted access” pilot 为经过审核的 security organisations（成熟 incident-response firms、government）提供 capability access 用于 defense。如果 pilot scale，access asymmetry 有利于 defenders。

### Nuclear / 核领域

公开文档中四个 CBRN domains 里分析最少的一类。Threat model 不同：fissile-material acquisition 主导难度，而不是 information。AI 在 information layer 上的 uplift 对 novice 实践帮助有限。2024-2025 major-lab reports 没有识别 nuclear-specific threshold crossing。

### Novice-relative vs expert-absolute / 新手相对与专家绝对

四个 domains 中的共同模式：

- **Novice-relative uplift。** 高。Multiplicative。Anthropic 2025 bio 中为 2.53x。
- **Expert-absolute capability。** ceiling 高。Expert 比 novice 能提取更多，因为 expert 知道问什么、如何解释。

Safety cases 的含义：只处理 novice uplift（通过 input filters、refusals、uncertainty）不足以控制 expert-absolute capability。还需要额外措施：elicitation-hardening、capability unlearning（Lesson 17）和 control protocols（Lesson 10）。

### Cross-domain synthesis / 跨领域综合

| Domain | 2024 | 2025 | Inflection |
|---|---|---|---|
| Bio | mild uplift | 2.53x uplift, ASL-3 approach | acquisition-phase automation |
| Chem | mild uplift | execution-gap erosion via vision | real-time wet-lab correction |
| Cyber | code assistance | 80-90% campaign automation | agentic coding |
| Nuclear | limited | limited | material-access bottleneck holds |

三个 domains 跨过 thresholds。一个仍被 non-informational barriers 限制。

## Exercises / 练习

1. 阅读 Anthropic 2025 年 11 月 cyber report。枚举 4-6 个 human-intervention steps，并论证下一代模型最先会自动化哪一个。

2. Chem/bio execution gap 正通过 vision 侵蚀。设计一个测量 tacit-knowledge uplift 的 evaluation，同时不跨越 ITAR/EAR 边界。

3. Nuclear uplift 看起来受 material access 限制。分别支持和反对 “未来 AI breakthrough 可以改变这个 bottleneck” 的立场。

4. 为 cyber-capable frontier model 构造一个 safety case（Lesson 18 three-pillar），同时限制 novice 与 expert uplift。

5. 从四个 domains 中选一个，根据 2024-2025 轨迹写一段 2027 forecast。指出什么 evidence 会 falsify 你的 forecast。

## Key Terms / 关键术语

| 术语 | 常见说法 | 实际含义 |
|------|-----------------|------------------------|
| Uplift | “AI helps attackers” | 可归因于 AI assistance 的 attacker capability increase |
| Novice-relative uplift | “multiplicative” | AI 相对 status-quo 对 novice 的帮助程度 |
| Expert-absolute capability | “ceiling” | expert 能从模型提取的最大 capability |
| Execution gap | “doing vs knowing” | 历史防御：tacit wet-lab skill 阻挡 novices |
| Agentic coding | “autonomous attacks” | multi-step autonomous cyber-task execution |
| Acquisition phase | “pre-synthesis steps” | bio threat 的 procurement、equipment、permit stages |
| Trusted access | “defender-only pilot” | OpenAI 2025 program，为 vetted defenders 提供 capability access |
